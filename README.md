

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/afthab8990/gsap-task.git
   cd Next_gsap_task/my-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Required Dependencies**

   The project requires the following dependencies (should be in your `package.json`):

   ```json
   {
     "dependencies": {
       "react": "^18.0.0",
       "react-dom": "^18.0.0",
       "next": "^13.0.0",
       "gsap": "^3.12.0"
     },
     "devDependencies": {
       "tailwindcss": "^3.3.0",
       "typescript": "^5.0.0",
       "@types/node": "^20.0.0",
       "@types/react": "^18.0.0"
     }
   }
   ```

## Running the Project

1. **Development Mode**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
